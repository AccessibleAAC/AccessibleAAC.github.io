AAC Hub Web Tool

Please note that this tool requires a browser supports the Web Serial API . The API currently experimental and is only supported by recent versions of the Chrome, Opera or Edge browsers. To use this tool you need to enable your browser's Experimental Web Platform Features by copying and pasting the corresponding URL into your browser:

  • chrome://flags/#enable-experimental-web-platform-features
  • opera://flags/#enable-experimental-web-platform-features
  • edge://flags/#enable-experimental-web-platform-features

This tool is still a work in progress. Reloading the page and reconnecting may be required, as some aspects of the API are not yet handled gracefully by the tool.
